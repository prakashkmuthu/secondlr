# Linear Regression Salary Prediction

## Project Overview

This project predicts salary based on years of experience using Linear Regression.

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- FastAPI
- Streamlit
- Git
- Docker
- AWS EC2

## Project Structure

data.csv
train.py
model.pkl
main.py
app.py
requirements.txt

## Install Dependencies

pip install -r requirements.txt

## Train Model

python train.py

## Run FastAPI

uvicorn main:app --reload

## Run Streamlit

streamlit run app.py
